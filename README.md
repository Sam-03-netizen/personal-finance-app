# 💸 Personal Finance & Expense Analytics App

A modern **React-based Personal Finance Tracker** that helps users manage their income, expenses, budgets, and financial insights through a clean and interactive dashboard.

---

## 🚀 Features

### 📊 Dashboard
- Total Income
- Total Expenses
- Net Balance
- Top Spending Category
- Spending by Category Pie Chart

### 💳 Transactions
- Add new income or expense transactions
- View all transactions in a clean card-based layout
- Delete transactions
- Search transactions by title or notes
- Filter transactions by:
  - Category
  - Type (Income / Expense)

### 🎯 Budget Management
- Set a monthly budget
- Track total spending
- View remaining budget
- Visual budget progress bar
- Over-budget warning

### 📈 Analytics
- Total Transactions
- Recurring Transactions Count
- Average Expense
- Income vs Expense Bar Chart

---
